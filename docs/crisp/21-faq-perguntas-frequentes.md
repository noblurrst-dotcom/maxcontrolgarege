# Perguntas Frequentes (FAQ)

## Conta e Login

**Como mudo meu email de login?**
Atualmente o email é fixo. Para mudança, contate o suporte.

**Como mudo minha senha?**
Faça login → menu do perfil (canto superior direito) → opção de senha. Ou use "Esqueci minha senha" na tela de login.

**Posso ter várias empresas em uma conta?**
Não. Cada conta = uma empresa. Para múltiplas empresas, crie contas separadas.

**Esqueci a senha — como recupero?**
Tela de login → "Esqueci minha senha" → digite email → clique no link enviado por email.

## Vendas e Pagamentos

**Como dou desconto em uma venda?**
No formulário, no campo Desconto, escolha "valor" (R$) ou "percentual" (%).

**Como registro pagamento parcial?**
Abra a venda → "Capturar pagamento" → digite só o valor pago. O status vira "parcial". Ao receber o restante, capture mais um pagamento.

**Como cancelo uma venda?**
Abra a venda → "Cancelar venda" → confirme. Pagamentos vão sair do financeiro automaticamente.

**Como faço uma cortesia (gratuito)?**
Abra a venda → "Marcar como cortesia". Não conta no faturamento.

**Posso parcelar no crédito?**
Sim. Ao capturar pagamento com forma "crédito", informe o número de parcelas (1 a 12).

**Preciso de máquina de cartão integrada?**
Não. O sistema apenas registra o pagamento. A máquina física é separada.

## Agenda

**Como evito conflito de horário?**
Veja no calendário antes de marcar. O sistema não bloqueia automaticamente.

**Cliente desmarcou — o que faço?**
Abra o agendamento → mude status para "cancelado" ou exclua.

**Cliente remarcou — preciso refazer?**
Não. Edite o agendamento existente e mude data/hora.

**Posso ter colunas separadas por funcionário?**
Não nativamente. Use cores diferentes por funcionário para organização visual.

## Clientes e Veículos

**Como uso o sistema sem cliente cadastrado?**
Em vendas, você pode digitar o nome livre sem selecionar cliente. Mas o ideal é cadastrar para histórico.

**Cliente trocou de veículo — como atualizo?**
Edite o cadastro do cliente → seção Veículos → adicione o novo / remova o antigo.

**Como vejo histórico de um cliente?**
Abra o cadastro do cliente. Há seção mostrando vendas, agendamentos e total gasto.

**O sistema avisa aniversariantes?**
Tem filtro de aniversariantes do mês em Clientes. Notificação automática não há ainda.

## Financeiro

**Diferença entre entrada e venda?**
Venda é o ato de vender. Entrada no financeiro é o dinheiro entrando. Ao capturar pagamento de uma venda, é gerada uma entrada automaticamente.

**Posso ter mais de uma conta bancária?**
Sim. Cadastre quantas precisar em Financeiro → Contas bancárias.

**Como vejo saldo de cada conta?**
Em Financeiro → Contas. Cada conta mostra saldo atualizado conforme entradas/saídas.

**O sistema concilia com extrato bancário?**
Não automaticamente. Lançamentos são manuais (ou automáticos via venda).

## Equipe

**Sub-usuário pode ver financeiro?**
Apenas se você der permissão "ver" no módulo Financeiro para ele.

**Funcionário vê dados de outro funcionário?**
Vê os dados que tem permissão. Se tem acesso a Vendas, vê todas as vendas (não filtra por autor automaticamente).

**Como cadastro funcionário CLT vs PJ?**
No cadastro de Colaboradores, escolha o "tipo" (CLT, Freelancer PJ, Freelancer Autônomo). Os campos mudam conforme.

**Comissão é automática?**
O cálculo é automático com base no % do colaborador, mas o pagamento da comissão é lançado manualmente em Pagamentos a Colaboradores.

## Vitrine Digital

**Como divulgo minha vitrine?**
Pegue o link em Vitrine Digital → cole em Instagram, WhatsApp, Google Meu Negócio, panfletos.

**Cliente agenda direto e cai na agenda?**
A solicitação cai numa lista de aprovação. Você aprova → vai para a agenda. Não aprova automaticamente para evitar conflitos.

**Posso esconder a vitrine temporariamente?**
Sim. Em Vitrine Digital, desative o interruptor "Vitrine ativa".

## Mobile / App

**Funciona no celular?**
Sim. Use o navegador ou instale como app (PWA).

**Funciona offline?**
Sim. Sincroniza quando voltar a internet.

**Onde baixo o app?**
Não tem na Play Store/App Store. É instalação direto pelo navegador (PWA). Veja o tópico "Aplicativo / Mobile".

## Dados e Backup

**Onde meus dados ficam armazenados?**
Em servidores na nuvem. Acessíveis de qualquer dispositivo com seu login.

**Os dados estão seguros?**
Sim. Cada conta tem isolamento total. Senhas são criptografadas. Conexão por HTTPS.

**Posso exportar meus dados?**
Em Relatórios é possível exportar PDF/CSV de várias visões. Exportação completa do banco — contate o suporte.

**Se eu cancelar a conta, perco os dados?**
Há prazo de retenção. Antes de cancelar, exporte o que precisar.

Veja o tópico **Suporte**.
