# A.T.A Gestão — Visão Geral

## O que é

O A.T.A Gestão é um sistema completo de gestão para estéticas automotivas, oficinas e centros automotivos. Funciona pelo navegador (web) e também pode ser instalado como aplicativo no celular (PWA).

## Para quem é

- Estéticas automotivas
- Oficinas mecânicas e centros automotivos
- Lava-rápidos premium
- Polidoras e funilarias
- Profissionais autônomos do setor automotivo

## Módulos disponíveis

O sistema é organizado em módulos. Cada usuário pode ver os módulos que tem permissão:

- **Painel (Dashboard)** — visão geral do negócio (vendas, faturamento, agendamentos do dia)
- **Vendas** — registrar vendas, criar orçamentos, controlar pagamentos
- **Agenda** — agendar serviços, ver calendário diário/semanal/mensal
- **Clientes** — cadastro de clientes e seus veículos
- **Serviços** — catálogo de serviços oferecidos com preços
- **Financeiro** — controle de entradas, saídas, contas bancárias
- **Colaboradores** — equipe (CLT, PJ, autônomos), salários, comissões
- **Relatórios** — análises e exportações
- **Configurações** — identidade visual, dados da empresa, impostos
- **Vitrine Digital** — página pública onde clientes podem agendar online
- **Usuários** — criar acessos para a equipe com permissões diferentes

## Como acessar

1. Acesse pelo navegador (Chrome, Safari, Edge) no endereço fornecido
2. Faça login com email e senha
3. Use o menu superior para navegar entre os módulos

## Funciona offline?

Sim. O sistema é uma PWA — depois de instalar como aplicativo, você pode usá-lo mesmo sem internet. As alterações são sincronizadas automaticamente quando a conexão volta.

## Funciona no celular?

Sim. O sistema é totalmente responsivo, funciona em qualquer tamanho de tela. Para a melhor experiência no celular, recomendamos instalar como aplicativo (veja o tópico "Instalar como app").

## Preciso instalar algo?

Não. É 100% web. Basta abrir no navegador. Opcionalmente, você pode "instalar" como app no celular ou desktop para um atalho mais rápido.

## Quantos usuários posso ter?

Você (titular da conta) pode criar quantos sub-usuários quiser para sua equipe, cada um com permissões personalizadas (administrador, gerente, operador ou visualizador).

## Os dados ficam seguros?

Sim. Cada conta tem seus dados isolados. Ninguém vê os dados de outra empresa. As senhas são criptografadas e o acesso é controlado por sessão.
