# Aplicativo / Mobile / Offline

## O sistema tem aplicativo?

Sim, o A.T.A Gestão é uma **PWA** (Progressive Web App). Você pode instalar como aplicativo no celular ou desktop, sem precisar de loja de apps.

## Como instalar no Android

### Pelo Chrome
1. Abra o sistema no Chrome
2. Aparece um banner "Adicionar à tela inicial" (ou clique no menu ⋮ → **Instalar app**)
3. Confirme
4. O ícone aparece na tela inicial como um app normal

### Pelo Samsung Internet
1. Abra o sistema
2. Menu ☰ → **Adicionar página a** → **Tela inicial**
3. Confirme

## Como instalar no iPhone (iOS)

⚠️ No iOS é só pelo **Safari** (Chrome no iPhone não permite PWA).

1. Abra o sistema no **Safari**
2. Toque no botão **Compartilhar** (quadrado com seta para cima)
3. Role e toque em **Adicionar à Tela de Início**
4. Confirme o nome
5. O ícone aparece na tela inicial

## Como instalar no computador (Windows/Mac)

### Chrome ou Edge
1. Abra o sistema
2. Veja o ícone de **+ Instalar** na barra de endereço (lado direito)
3. Clique e confirme
4. Vira um app no menu Iniciar (Windows) ou Aplicativos (Mac)

## Funciona sem internet

Sim. Após o primeiro carregamento, o sistema funciona offline:
- Você pode **ver** dados já carregados
- Pode **criar/editar** vendas, agendamentos, clientes
- Quando a internet voltar, **sincroniza automaticamente**

## Limitações offline

- Não dá pra fazer login pela primeira vez offline
- Algumas funções específicas podem requerer internet (envio de emails, geração de PDF complexo, vitrine)
- Fotos podem demorar mais para sincronizar dependendo do tamanho

## Como sei que está sincronizado

O sistema tenta sincronizar automaticamente em segundo plano. Se houver dados pendentes de sync, geralmente há um indicador. Conexão estável = sync rápido.

## Os dados ficam no celular?

Sim, tem cópia local (cache) para funcionar offline. A fonte primária é o servidor (nuvem). Mesmo se você apagar o app, os dados ficam salvos na sua conta.

## App lento ou desatualizado

Se notar que o app está com versão antiga:
1. Faça um **hard reload** (Ctrl+Shift+R no Windows / Cmd+Shift+R no Mac)
2. No celular, force fechar o app e reabrir
3. Em último caso, desinstale e instale de novo (não perde dados — estão na nuvem)

## Atualizações automáticas

A PWA atualiza automaticamente. Quando há nova versão, na próxima vez que você abrir o app, a versão nova é baixada em segundo plano e ativada.

## Posso usar no tablet

Sim, funciona perfeitamente em tablets (Android e iPad). Layout responsivo se adapta.

## Tira foto direto pelo app

Sim. Em checklists e cadastros de veículo, ao adicionar foto, você pode escolher entre:
- Câmera (tira foto na hora)
- Galeria (selecionar imagem existente)

Útil pra documentar veículos no balcão.

## Notificações push

Notificações push estão em desenvolvimento ou disponíveis dependendo da versão. Verifique nas Configurações se há essa opção.

## App ocupa muito espaço?

Não. PWA é leve — geralmente menos de 10 MB de cache + suas fotos/dados.

## Como desinstalar

### Android
Pressione e segure o ícone → Desinstalar

### iOS
Pressione e segure o ícone → Remover app

### Desktop
Abra o app → menu ⋯ → **Desinstalar**

Desinstalar não apaga seus dados — eles continuam na nuvem.
