# Clientes

## O que é

O módulo Clientes é o cadastro de todos os seus clientes. Cada cliente pode ter um ou mais veículos vinculados, histórico de gastos e fotos.

## Como acessar

Menu superior → **Clientes**.

## Como cadastrar um novo cliente

1. Clique em **+ Novo cliente**
2. Preencha os dados:
   - **Nome** (obrigatório)
   - **Telefone** (importante para contato)
   - **Email**
   - **CPF/CNPJ**
   - **Endereço**
   - **Aniversário** (para você lembrar de mandar parabéns)
   - **Observações** (qualquer detalhe extra)
3. Adicione veículo(s) — placa, modelo, marca, cor, ano
4. Opcionalmente adicione fotos do veículo (4 ângulos: frente, traseira, lateral direita, lateral esquerda)
5. Clique em **Salvar**

## Como editar um cliente

1. Clique sobre o nome do cliente na lista
2. Faça as alterações
3. Clique em **Salvar**

## Como excluir um cliente

1. Abra os detalhes do cliente
2. Clique em **Excluir**
3. Confirme

⚠️ **Atenção**: ao excluir um cliente, o histórico de vendas associado a ele é mantido (com o nome registrado), mas o cadastro some.

## Como buscar um cliente

Use a barra de busca no topo da lista. Você pode buscar por:
- Nome
- Telefone
- Placa do veículo
- CPF/CNPJ
- Email

A busca é instantânea conforme você digita.

## Total gasto pelo cliente

Cada cliente tem um campo **Total gasto** que soma automaticamente todas as vendas pagas dele. Útil para identificar seus melhores clientes.

## Aniversariantes do mês

O sistema permite filtrar clientes pelo mês de aniversário. Use isso para campanhas promocionais ou cumprimentos personalizados.

## Posso enviar mensagem direta pelo WhatsApp?

Sim. Na lista de clientes, há um botão de WhatsApp ao lado do telefone. Clique e abre direto o WhatsApp com o número do cliente preenchido.

## Posso importar clientes de uma planilha?

Atualmente o cadastro é manual. Se você tiver muitos clientes para importar, contate o suporte.

## Veículos e fotos do veículo

Veja o tópico específico **Veículos** para detalhes sobre cadastro de veículos e fotos.
